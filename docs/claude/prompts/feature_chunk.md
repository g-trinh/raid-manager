Read the @docs/features/v2/components folder. It contains all components for the first minimal gaming loop.

As a product manager, you have to split the gaming loop into smaller chunks that can be handled in half a day maximum by a team.
In each team, each agent will intervine in this specific order :
1. product designer, if needed
2. software architect
3. game engine developer, if needed
4. game script developer, if needed
5. backend developer, if needed
The task must contain all necessary feature information to :
- produce screens mockups
- design software architecture
- implement the feature

The team will then split the task into technical tasks, following Team organization instructions.

You will follow the @docs/claude/templates/task.md template for each new task.

Don't :
- Dive into game, UI or UX design
- Dive into software architecture
- Implement the feature
